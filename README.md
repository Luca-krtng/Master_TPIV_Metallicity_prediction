Stellar Photometry and Calibration ProjectThis repository contains a collection of Jupyter Notebooks and data files focused on stellar parameter visualization, flux calculations, and the creation of calibration plans for MS (Main Sequence) and RG (Red Giant) stars using various survey data (SDSS, PS1, Gaia, and Euclid).📊 Project OverviewThe project involves:Visualizing stellar parameters from the AMBRE synthetic dataset.Calculating fluxes and magnitudes for pseudo color-color diagrams.Developing calibration plans for stellar selection.Comparing observed data against theoretical isochrones and literature (e.g., Starkenburg et al. 2017).📂 File StructureJupyter Notebooks (Analysis & Plotting)plot-stellar-param.ipynb: Visualizes stellar parameters from the AMBRE synthetic dataset.Starkenburg_paper-diagram.ipynb: Reproduces the pseudo color-color diagram for PS1 and g-i color from Starkenburg et al. (2017).different-surveyv2.ipynb: Plots pseudo color-color diagrams for g-i and g-r for PS1, SDSS, and Gaia.Euclid-photometry.ipynb: Generates pseudo color-color diagrams specifically for Euclid photometry.Luminosity-determination.ipynb: Uses isochrones to select MS and RG stars and performs polynomial fitting for pseudo color-color diagrams.Calibration-plan.ipynb: Creates the actual calibration plans for MS and RG stars using the polynomial fits.Observation-determination.ipynb: Overlays training samples onto the calibration plans and calculates error statistics.PS1-invistigation.ipynb: Investigates the separability of surface gravity ($\log(g)$).Data & Resourcesobserved-data.zip: Contains the observed data training sample.isochrones.zip: Theoretical isochrones used for star selection and calibration.filter.zip: Filter information used for flux calculations (SDSS, PS1, Gaia, Euclid).Euclid_magnitude.parquet: Saved fluxes and magnitudes for Euclid.calculated_magnitudes.parquet: Pre-calculated data for PS1 and SDSS to skip redundant flux calculations.calibration_grids.npz.zip: Compressed calibration grids.🔗 External ResourcesFilters used in this project are also available via the Spanish Virtual Observatory (SVO):Filter Profile Service.
